# Simple CRUD CPP
Only in Linux, Debian distribution

Prerequisite:  
`sudo apt install cmake libboost-dev -y`

1. Install MySQL-Connector-CPP
`sudo apt install libmysqlcppconn-dev -y`

2. Create build  
```
  mkdir build && cd build
  cmake ..
  make
```

3. Run program
`./simple_cli_app`
